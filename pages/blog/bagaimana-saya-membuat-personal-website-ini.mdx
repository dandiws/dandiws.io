Sebagai seorang developer saya selalu ingin memiliki personal website yang terlihat menarik.
Dalam membangun personal website banyak sekali opsi teknologi yang dapat di gunakan.
Salah satunya adalah [JAMstack](https://jamstack.wtf/) yang sedang populer beberapa tahun belakangan.

Beberapa fitur yang akan saya implementasikan pada personal website saya antara lain adalah:

- Blog
- Project
- Mode gelap dan terang

## Memulai Project

Sebelum memulai project pastikan komputer anda telah terinstall Node.js dan NPM (atau yarn).
Disini saya menggunakan yarn. Untuk membuat project baru Next.js, jalankan perintah berikut:

```bash
yarn create next-app <nama-project>
```
